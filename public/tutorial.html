Welcome to Pycraft!

Overview

Pycraft is a research project to help investigate the psychology of programming. Particularly in age groups such as middle-schoolers, high-schoolers, and college students. Pycraft gathers the analytics of the different age groups and determines the differences in how they go about solving different programming problems.

Pycraft is composed of the following components:
  • Blockly
  • Python
  • Minecraft
  • Minecraft Forge
  • RaspberryJam Mods

Blockly

    What is it?
    Blockly is a library that adds a visual code editor to web and mobile apps. The Blockly editor uses interlocking, graphical blocks to represent code concepts like variables, logical expressions, loops, and more. It allows users to apply programming principles without having to worry about syntax or the intimidation of a blinking cursor on the command line.
    How did Pycraft use it?
    Blockly is the main UI component in Pycraft. Used for its visual programming capabilities.

Python

    What is it?
    Python is an interpreted, object-oriented, high-level programming language with dynamic semantics. Its high-level built in data structures, combined with dynamic typing and dynamic binding, make it very attractive for Rapid Application Development, as well as for use as a scripting or glue language to connect existing components together.
    How did Pycraft use it?
    While using symantics of the visual Blockly blocks, these generate syntactically correct user-generated Python scripts. These scripts are then saved and ran inside of Minecraft.

Minecraft

    What is it?
    An open-world game that promotes creativity, collaboration, and problem-solving in an immersive environment where the only limit is your imagination.
    How does Pycraft use it?
    Minecraft is one of the main driving componets of Pycraft. While by itself Minecraft has endless creative possibilites. Pycraft takes advantage of this and helps the users learn programming while playng Minecraft. The generated Python scripts are ran in the text box inside of Minecraft.

Minecraft Forge

    What is it?
    Is a free, open-source modding API and loader designed to simplify compatibility between community-created game mods in Minecraft: Java Edition. The end-user (player) must play the version of Minecraft that Forge was downloaded for before it can be used. For example, Forge 1.12 will need the 1.12 version of Minecraft in order to run correctly. Forge can then be used from the Play drop-down in the Minecraft launcher.
    How does Pycraft use it?
    The Minecraft Forge API is used to bridge the gap in connecting the RaspberryJam Mods to work inside of Minecraft.

RaspberryJam Mods

    What is it?
    Mods that use the Minecraft API and allow for Minecraft to be controlled by Python code.


Get Started

Components in Menu Bar:
  • File Name
  • Create Script
  • Save Blocks
  • Load Blocks
  • Create Snapshot
  • Restore Snapshot
  • Tutorial
    File Name
    Enter the name of your script. The following characters are prohibited (? : " / \ | * < >).

    Create Script
    After creating a filename click on 'Create Script' to save it in the script library.

    Save Blocks
    Saves current workspace to a file.

    Load Blocks
    Loads workspace from a file.

    Create Snapshot
    Saves current state of the workspace.

    Restore Snapshot
    Restores the previous snapshot and corresponding code.

    Tutorial
    Takes you to the tutorial page in a separate tab.

    Workspace
    This is the block view of the users work.

    Code
    This displays the code that has been created with the blocks.



Blocks

    Minecraft Blocks

      Modify Block
      Input: Takes block variable
      Output: New block type

      Set Block
      Sets a block at a position variable to a particular type.

      Set Blocks
      Set many blocks at a time, filling the gap between 2 sets of position variables.

      Get Block
      Returns block type located at position variable.

      Get Block Type and ID
      Returns the block type and corresponding ID.

      Get List of Block Types
      Returns all block IDs ranging from one position to another.

      Get Height
      Returns the y coordinate of an x/z coordinate which represents the highest (non-air) block.

    Player Blocks

      Get Player ID
      Returns the entity ID of a player.

      Get Player IDs
      Returns the entity IDs of all players connected to the game.

      Get Player Position
      Returns player's current position.

      Set Player Position
      Sets the location of a player to the position variable 'y'.

      Get Position Under Player
      Returns the position of the block under player.

      Get Players Direction
      Returns the direction of player as a position variable.

      Get Players Angle of Rotation
      Returns the rotation angle as a float variable.

      Get Players Pitch
      Returns the pitch angle as a float variable.

    Position Blocks

      Position
      Modifies the current position 'x' to a user defined position.

    Entity Blocks

      Get Other Players Position
      Returns the position of another player.

      Set Entity's Position
      Uses a position variable to set 'x' to 'y'.

      Get Position Under Player
      Returns the position underneath another player.

      Get Rotation of Another Player
      Returns the roation of another player as a float.

      Get Pitch of Another Player
      Returns the pitch of another player as a float.
